# 소스코드 설명
```
#include<iostream>
```
- cin, cout, 등 라이브러리 객체의 선언을 포함하고 있는 헤더파일인 'iostream' 을 포함하는 코드

```
int main()
```
- main 함수를 시작하는 코드

```
std::cout << "Hello";
```
- 삽입 연산자 함수로서 다음에 오는 문장을 화면에 출력해주는 객체이다. 즉 Hello를 출력하도록 하는 코드

```
return 0;
```
- 0을 반환하고 main 함수를 종료한다.
