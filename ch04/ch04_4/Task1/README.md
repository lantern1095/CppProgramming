
### string 객체는 문자열 입력 받을 때 몇 자를 입력할지 물어 보지않고 알아서 크기를 자동 조절한다. 동작 원리를 검색해보고 설명하라
- string 객체는 내부적으로 문자를 저장할 배열을 가지고 있고, 이 배열은 힙 메모리 영역을 사용하기 때문에 입력받은 문자열의 길이에 따라 동적으로 그 크기를 늘리거나 줄일 수 있다.

### 문자열을 입력할 때 마지막에 입력하는 엔터키와 getline 함수의 마지막 인자에 전달하는 구분문자와 차이를 설명하라
- 엔터키는 표준 입력 스트림에 내부적으로 \n 를 입력하고, getline의 구분 문자는 getline의 세 번째 인자로 전달되어 해당 문자가 나올 때까지만 문자열을 읽어들이겠다는 의미이다.
