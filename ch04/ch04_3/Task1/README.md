## 실습과제 1

### 메모리 누수를 설명하라.
- 동적 할당받은 메모리를 해제하지 않거나 동적으로 할당받은 메모리의 주소를 잃어버리게 된다면 사용하지 않는 객체들이 프로그램 실행 중에 힙 영역에 계속 남아있게 되는데, 이를 메모리 누수라 한다.  

### 메모리 누수가 왜 심각한 문제를 발생시키는지 설명하라.
-  반복적으로 사용하지 않는 공간이 힙 영역에 늘어나게 된다면 사용할 수 있는 메모리가 점점 줄에들게 됨을 의미하고, 끝내는 프로그램이 강제 종료되거나 프로그램의 성능이 심각하게 저하될 수 있다.

### 메모리 누수를 방지하는 방법을 설명하라.
- 동적으로 할당 받은 메모리는 사용 목적을 달성하면 반드시 반납해야 한다. (C++ 같은 경우에는 delete 로 해제를 해주어야 한다)
