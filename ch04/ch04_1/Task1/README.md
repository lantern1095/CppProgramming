# 실습과제1

## 배열명의 의미를 설명하라.
- 배열명은 배열의 첫 번째 요소의 주소를 의미하며 상수로 취급되기 때문에 임의로 수정할 수 없다.

## 객체 포인터의 의미를 설명하라.
- 객체의 주소를 저장하는 변수로서, 구조체 포인터와 유사성을 가진다. 하지만 객체 포인터라고 해서 객체인 것은 아니다. 객체 포인터는 객체의 주소값을 저장하는 포인터 변수이기 때문이다.

## 포인터를 이용하여 배열 각 원소의 주소를 구하는 방법을 설명하라
- `int *p = arr` 와 같이 포인터 변수 p를 선언하고 p에 산술 연산을 적용하여 배열의 주소를 하거나, `&arr[i]` (i는 임의의 정수) 로 주소값을 직접 반환한다.
  
## 포인터를 이용하여 배열 각 원소의 값을 구하는 방법을 설명하라
- `(p + i)->` 와 같이 객체 배열의 첫 번째 요소의 주소가 저장된 포인터 변수에 산술 연산을 적용하여 주소를 이동하며 각 객체의 멤버들에 직접 접근할 수 있다.
- 비슷한 방법으로, `p->` 로 접근하고 `p++`로 주소값을 증가시키는 방법도 있다.
