# 실습과제 1
## 구조체와 클래스의 차이점을 설명하라.
- 둘다 사용자 정의 자료형이지만, 구조체는 접근자가 기본 public 이고, 클래스는 private 이다.

## 클래스와 객체의 차이점을 설명하라.
- 사용자 정의 자료형으로서 객체를 구현하기 위한 멤버 변수와 멤버 함수의 집합으로, 설계도와 같은 역할을 한다.
- 클래스형 변수로서, 클래스를 이용해 만들어 낸 실체이다.

## sizeof(클래스명)의 결과값을 출력해보고 결과를 설명하라.
- 객체에 할당된 메모리의 크기값 (바이트) 을 반환한다. 멤버 함수는 클래스 내부에서 메모리를 할당받지 않는다.
