# 실습과제1

## 객체의 생성과 소멸의 의미를 설명하라
- 객체 생성의 의미는 운영체제가 관리하는 메모리 공간을 프로그램에서 할당받아 사용할 수 있게 됨을 의미한다.
- 객체 소멸의 의미는 프로그램에서 사용하던 메모리 공간을 운영체제에 반납하고, 더 이상 해당 공간을 사용할 수 없게 됨을 의미한다.

## 지역변수(지역객체), 전역변수(전역객체)의 차이를 조사하라, 선언위치, 접근범위, 생성시점, 소멸시점, 초기화 여부 등의 관점에서 비교하라
|   |지역변수(지역객체)|전역변수(전역객체)
|---|---|---|
|선언위치|함수 내부 |함수 외부|
|접근범위|해당 함수 내에서만|프로그램 어디에서나|
|생성시점|해당 함수가 호출되었을때|프로그램 시작 시(main() 이전)|
|소멸시점|해당 함수를 벗어나기 직|프로그램이 종료되기 직전|
|초기화 여부|초기화 필요(안할시 쓰레기값이 들어감)|자동으로 0으로 초기화된다|

## 프로그램 실행 후 main함수가 실행되기 전에 어떤 일이 처리되는지 조사해보라
- 전역변수의 선언 및 초기화 과정과, 전역객체가 생성되며 생성자를 호출한다. 이후 main()함수를 실행한다

## main함수가 종료 후 프로그램이 종료하기 전에 어떤 일이 처리되는지 조사하라
- 전역객체의 소멸자를 호출하여 사용하던 메모리를 운영체제에 반납하고 프로그램을 종료한다.
