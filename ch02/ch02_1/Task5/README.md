# 소스코드 설명

`#include <iostream>`
- cin, cout, 등 라이브러리 객체의 선언을 포함하고 있는 헤더파일인 'iostream' 을 포함하는 코드

`using namespace std;`
- using 지시어를 사용하여 std 네임스페이스에 선언된 이름들을 사용할 때 std::를 생략한다.

`int main()`
- main 함수 시작

`for (int n = 0; n < 4; n++)`
- 변수 n을 선언하고 0으로 초기화한 뒤, 3이 될 때까지 반복한다.
 
`for (int m = 0; m <= n; m++)`
- 변수 m을 선언하고 0으로 초기화한 뒤, n의 값이 될 때까지 반복한다.

`cout << "*";`
- cout 객체를 사용하여 화면에 `*`를 출력한다.

`cout << "\n";`
- cout 객체를 사용하여 줄바꿈을 한다.

`return 0;`
- 0을 반환하고 메인 함수를 종료
